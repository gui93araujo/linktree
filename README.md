# Linktree

Projeto inspirado no Linktree para centralizar links pessoais.

## 🚀 Tecnologias

- React
- TypeScript
- TailwindCSS
- Vite
- Firebase Authentication
- firebase firestore

## 📌 Funcionalidades

- ➕ Criar links sociais
- 📱Layout responsivo
- 🎨 Estilização com TailwindCSS
- 🔐 Autenticação de usuário com Firebase

## ▶️ Rodando o projeto

```bash
npm install
npm run dev
```
